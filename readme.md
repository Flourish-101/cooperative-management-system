# Cooperative Management Information System (CMIS)

A web-based Cooperative Management Information System designed to digitize and simplify the operations of a cooperative society.

## Project Goals

The system aims to:

- Manage cooperative members
- Track member contributions
- Record meeting attendance
- Process loan requests and approvals
- Monitor loan repayments
- Generate administrative reports

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- NestJS
- TypeScript

### Database
- PostgreSQL
- Prisma ORM
